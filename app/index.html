    Herinnering

    Elke dag wil Dokter Tommy je vragen wat je van de dag vind.
    Wil je dat? Zo ja, zet herinnering op aan en accepteer notificaties.

    Je kunt de herinnering altijd op uit zetten.

    Oops

    DokterTommy heeft notificaties nodig om je een herinnering te kunnen sturen. Wil je een herinnering krijgen? Zet dan in de browser settings notificaties aan voor https://doktertommy.nl/.
    :

    Backup

    Je maakt een backup van je data door de onderstaande data te kopieren en deze in een tekst bestand te bewaren. De backup kun je terugzetten door de data in het data tekstveld te plakken.
